# História da Filosofia - Landing Page

Landing page estática para venda do guia em PDF **História da Filosofia**.

## Como configurar o checkout

No arquivo `index.html`, substitua:

```js
const CHECKOUT_URL = "https://SEU-LINK-DA-CAKTO-AQUI";
```

pelo link real do checkout da Cakto.

## Como publicar no GitHub Pages

1. Abra o repositório no GitHub.
2. Vá em **Settings > Pages**.
3. Em **Build and deployment**, selecione **Deploy from a branch**.
4. Escolha a branch `main` e a pasta `/root`.
5. Salve.
